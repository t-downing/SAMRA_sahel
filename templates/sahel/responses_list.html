Ce page présente les résultats de quelques éléments clés du modèle pour chaque réponse testée, avec les coûts totaux pour chaque réponse. On peut l'utiliser pour comparer comment les interventions humanitaires qui font partie de chaque réponse influencent ces éléments clés, et la rapport de coût-efficacité de chaque réponse.

Les résultats sont présentés seulement pour un scénario à la fois, alors le menu déroulant peut être utilisé pour voir les résultats pour un scénario différent.
